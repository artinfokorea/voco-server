# User Flow

1. 회원가입(레벨, 카테고리(복수,선택))
2. 스케쥴 생성(요읿별 시간)
3. 통화시작(스케쥴에 해당하는 시간 or 앱에서 사용자가 즉시 시작)
4. 콜 내역 리스트 조회
5. 콜 내역중 상세 조회(분석 결과 - 대화내용(패러프레이즈, 틀린 문법, 추천 문장 등), 분석결과, 점수)

## 유저

- 회원가입
    - 이메일, 비밀번호, 레벨, 카테고리(복수)
- 정보 수정

## 인증/ 인가

- 로그인 (access - 30분, refresh - 7일)
- 토큰 리프레쉬

## 스케쥴

- 스케쥴 조회
- 스캐쥴 생성 (요일별로 한타임, ex - 월,수,금 08:20 / 화목 09:00)
- 스케쥴 수정
- 스케쥴 삭제

## 통화

- 콜 내역 조회(목록)
- 콜 분석 상세 확인
